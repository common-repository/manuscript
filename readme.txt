Japanese style manuscript paper (for writing, not music) format on your entry.

== Description ==

This plugin adds a special tag to format entry text to manuscript style. This designed for especially Japanese characters. Vertical writing supported.

See screen shot section for format detail

== Installation ==

Place file in your /wp-content/plugins/ directory
and activate through the administration panel.

Usase, see my website http://www.vjcatkick.com/?page_id=7313
